# GitHub Finder

Minimalistic web app that allows the user to search for user accounts in the GitHub directory. Just start typing in the search box and the app will immediately provide you with suggestions about potential users that you are trying to find. The app also provides the storage and display of recent searches for quick and easy access to users that were previously searched.

Made with:

- React with TypeScript
- Tanstack Query for the handling of fetching and caching of data from the GitHub API
- localStorage web API for storage of recent searches
- plain, old CSS
- bundled and built with Vite
